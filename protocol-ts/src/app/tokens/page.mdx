## Colour Tokens {{ id: 'colour-tokens' }}

### Brand Colours

```css
:root {
  /* Primary Brand Colours */
  --color-yellow-500: #FFCB05;  /* Base brand yellow */
  --color-yellow-600: #E6B800;  /* Hover state */
  --color-yellow-700: #B45309;  /* Active state */

  --color-black-900: #171717;   /* Brand black */
  --color-black-950: #0F0F0F;   /* Deep black */
  --color-black-1000: #000000;  /* Pure black */
}
```

### Semantic Colours (Traffic Light)

```css
:root {
  /* Success - Green */
  --color-success-500: #22C55E;
  --color-success-600: #16A34A;  /* 4.5:1 contrast */
  --color-success-700: #15803D;  /* 7:1 contrast */

  /* Warning - Orange */
  --color-warning-500: #F97316;
  --color-warning-600: #EA580C;  /* 4.5:1 contrast */
  --color-warning-700: #C2410C;  /* 7:1 contrast */

  /* Error - Red */
  --color-error-500: #EF4444;
  --color-error-600: #DC2626;    /* 4.5:1 contrast */
  --color-error-700: #B91C1C;    /* 7:1 contrast */

  /* Info - Blue */
  --color-info-500: #3B82F6;
  --color-info-600: #2563EB;     /* 4.5:1 contrast */
  --color-info-700: #1D4ED8;     /* 7:1 contrast */
}
```

## Typography Tokens {{ id: 'typography-tokens' }}

### Font Families

```css
:root {
  --font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  --font-mono: 'JetBrains Mono', 'Fira Code', 'Courier New', monospace;
}
```

### Font Weights

```css
:root {
  --font-weight-normal: 400;
  --font-weight-medium: 500;
  --font-weight-semibold: 600;
  --font-weight-bold: 700;
  --font-weight-extrabold: 800;
  --font-weight-black: 900;
}
```

### Font Sizes

```css
:root {
  --font-size-xs: 0.75rem;      /* 12px */
  --font-size-sm: 0.875rem;     /* 14px */
  --font-size-base: 1rem;       /* 16px */
  --font-size-lg: 1.125rem;     /* 18px */
  --font-size-xl: 1.25rem;      /* 20px */
  --font-size-2xl: 1.5rem;      /* 24px */
  --font-size-3xl: 1.875rem;    /* 30px */
  --font-size-4xl: 2.25rem;     /* 36px */
  --font-size-5xl: 3rem;        /* 48px */
  --font-size-6xl: 3.75rem;     /* 60px */
}
```

### Line Heights

```css
:root {
  --line-height-none: 1;

  --line-height-normal: 1.5;    /* UI elements */
```

## Spacing Tokens {{ id: 'spacing-tokens' }}

```css

  --space-16: 4rem;     /* 64px */
```

## Implementation {{ id: 'implementation' }}

### Complete Token Set

Here's a complete, production-ready CSS file with all tokens:

```css
:root {
  /* === BRAND COLOURS === */
  --color-yellow-500: #FFCB05;
  --color-yellow-600: #E6B800;
  --color-yellow-700: #B45309;
  --color-black-900: #171717;

  /* === SEMANTIC COLOURS === */
  --color-success-600: #16A34A;
  --color-warning-600: #EA580C;
  --color-error-600: #DC2626;
  --color-info-600: #2563EB;

  /* === TYPOGRAPHY === */
  --font-primary: 'Inter', sans-serif;
  --font-weight-normal: 400;
  --font-weight-medium: 500;
  --font-weight-semibold: 600;
  --font-weight-bold: 700;
  --font-weight-extrabold: 800;
  --font-weight-black: 900;

  /* === SPACING === */
  --space-2: 0.5rem;    /* 8px */
  --space-4: 1rem;      /* 16px */
  --space-6: 1.5rem;    /* 24px */
  --space-8: 2rem;      /* 32px */

  /* === BORDER RADIUS === */
  --radius-md: 0.5rem;  /* 8px */
  --radius-lg: 0.75rem; /* 12px */
  --radius-xl: 1rem;    /* 16px */

  /* === SHADOWS === */
  --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
  --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1);
  --shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1);
  --shadow-xl: 0 20px 25px -5px rgb(0 0 0 / 0.1);
}
```

### Usage Example

```css
.button-primary {
  background-color: var(--color-yellow-500);
  color: var(--color-black-900);
  font-family: var(--font-primary);
  font-weight: var(--font-weight-semibold);
  padding: var(--space-3) var(--space-6);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-md);
}

.button-primary:hover {
  background-color: var(--color-yellow-600);
}
```

### Tailwind Configuration

If you're using Tailwind CSS, add these to your config:

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        yellow: {
          500: '#FFCB05',
          600: '#E6B800',
          700: '#B45309',
        },
        black: {
          900: '#171717',
        },
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
}
```
