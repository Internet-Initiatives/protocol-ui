## Best Practices {{ id: 'best-practices' }}

### Keyboard Navigation

<Properties>
  <Property name="Tab Order">
    Logical tab order matching visual layout
  </Property>
  <Property name="Focus Indicators">
    2px ring, clearly visible in all themes
  </Property>
  <Property name="Skip Links">
    Allow users to skip to main content
  </Property>
  <Property name="Keyboard Shortcuts">
    Document all keyboard shortcuts
  </Property>
</Properties>

### Screen Readers

<Properties>
  <Property name="Semantic HTML">
    Use proper heading hierarchy (h1-h6)
  </Property>
  <Property name="ARIA Labels">
    Meaningful labels for all interactive elements
  </Property>
  <Property name="Alt Text">
    Descriptive alt text for all images
  </Property>
  <Property name="Live Regions">
    Announce dynamic content changes
  </Property>
</Properties>

### Touch Targets

<Properties>
  <Property name="Minimum Size">
    44x44px for all interactive elements
  </Property>
  <Property name="Spacing">
    8px minimum between touch targets
  </Property>
  <Property name="Feedback">
    Visual feedback on touch/click
  </Property>
</Properties>
