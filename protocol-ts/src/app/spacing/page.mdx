          Small gaps between related items (list items, form field groups)
        </div>
      </div>
      <div className="px-6 py-4">
        <div className="font-semibold text-zinc-900 dark:text-white">md (16px)</div>

          Standard spacing for most UI elements (padding in cards, gaps between components)
        </div>
      </div>
      <div className="px-6 py-4">
        <div className="font-semibold text-zinc-900 dark:text-white">lg (24px)</div>

          Section spacing within containers (between content blocks in a card)
        </div>
      </div>

        <div className="font-semibold text-zinc-900 dark:text-white">xl (32px)</div>

          Major section separations (between cards, content sections)
        </div>
      </div>

## Code Reference {{ id: 'code-reference' }}

### CSS Custom Properties

```css
:root {
  --space-xs: 0.25rem;  /* 4px */
  --space-sm: 0.5rem;   /* 8px */
  --space-md: 1rem;     /* 16px */
  --space-lg: 1.5rem;   /* 24px */
  --space-xl: 2rem;     /* 32px */
  --space-2xl: 3rem;    /* 48px */
  --space-3xl: 4rem;    /* 64px */
  --space-4xl: 6rem;    /* 96px */
  --space-5xl: 8rem;    /* 128px */
}
```
