## Colour System {{ id: 'colour-system' }}

### Primary Brand Colours

```css
/* Brand Yellow - Primary Accent */
--color-yellow-500: #FFCB05;  /* CTAs, highlights, accents */
--color-yellow-600: #D97706;  /* Hover state */

/* Brand Black - Primary Text & Backgrounds */
--color-black-950: #0a0a0a;  /* Darkest background, dark mode */
--color-black-900: #171717;  /* Headings, premium backgrounds */
--color-black-800: #262626;  /* Body text on light */

/* Brand White */
--color-white: #FFFFFF;      /* Backgrounds, text on dark */
```

### Black Scale (Neutral Grays)

```css
--color-black-950: #0a0a0a;  /* Darkest */
--color-black-900: #171717;  /* Headings */
--color-black-800: #262626;  /* Body text */
--color-black-700: #404040;  /* Secondary text */
--color-black-600: #525252;  /* Tertiary text */
--color-black-500: #737373;  /* Disabled text */
--color-black-400: #a3a3a3;  /* Placeholder */
--color-black-300: #d4d4d4;  /* Borders */
--color-black-200: #e5e5e5;  /* Light borders */
--color-black-100: #f5f5f5;  /* Background tints */
--color-black-50: #fafafa;   /* Subtle backgrounds */
```

### Semantic Colours (Traffic Light System)

```css
/* Success - Green */
--color-green-600: #16a34a;  /* 4.54:1 contrast on white */
--color-green-800: #166534;  /* Text on light */
--color-green-100: #dcfce7;  /* Background tint */
--color-green-50: #f0fdf4;   /* Subtle background */

/* Warning - Amber */
--color-amber-600: #d97706;  /* 4.54:1 contrast on white */
--color-amber-800: #92400e;  /* Text on light */
--color-amber-100: #fef3c7;  /* Background tint */
--color-amber-50: #fffbeb;   /* Subtle background */

/* Error - Red */
--color-red-600: #dc2626;    /* 4.52:1 contrast on white */
--color-red-800: #991b1b;    /* Text on light */
--color-red-100: #fee2e2;    /* Background tint */
--color-red-50: #fef2f2;     /* Subtle background */

/* Info - Blue */
--color-blue-600: #2563eb;   /* 4.56:1 contrast on white */
--color-blue-800: #1e40af;   /* Text on light */
--color-blue-100: #dbeafe;   /* Background tint */
--color-blue-50: #eff6ff;    /* Subtle background */
```

### Critical Colour Rules

```text
NEVER USE:
- Yellow (#FFCB05) on white backgrounds for text
  Reason: 1.98:1 contrast fails WCAG at all levels

ALWAYS USE:
- Yellow (#FFCB05) ONLY for: CTA buttons, backgrounds (with dark text),
  highlights, accents, borders
- Black (#171717) text on yellow backgrounds
- Black (#262626) for body text on white
- White (#FFFFFF) text on black backgrounds

WCAG AA Compliance:
- Body text: Minimum 4.5:1 contrast
- Large text (24px+): Minimum 3:1 contrast
- UI elements: Minimum 3:1 contrast
```

## Typography {{ id: 'typography' }}

### Font Family

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

### Type Scale & Usage

```css
/* Headings */
--font-h1: 60px / 3.75rem;     font-weight: 900;  line-height: 1.2;
--font-h2: 48px / 3rem;        font-weight: 700;  line-height: 1.2;
--font-h3: 36px / 2.25rem;     font-weight: 700;  line-height: 1.2;
--font-h4: 30px / 1.875rem;    font-weight: 600;  line-height: 1.2;
--font-h5: 24px / 1.5rem;      font-weight: 600;  line-height: 1.2;
--font-h6: 20px / 1.25rem;     font-weight: 600;  line-height: 1.2;

/* Body Text */
--font-body: 18px / 1.125rem;  font-weight: 400;  line-height: 1.6;
--font-small: 14px / 0.875rem; font-weight: 400;  line-height: 1.5;

/* Letter Spacing */
--letter-spacing-headings: -0.02em;  /* For 48px+ headings */
--letter-spacing-normal: 0;          /* Body text */
```

### Font Weights

```text
400 Regular:    Body text, paragraphs, descriptions
500 Medium:     Emphasized text, light headings
600 Semibold:   Subheadings, labels, buttons
700 Bold:       Section headings, important text
800 Extrabold:  Page titles, hero text
900 Black:      Logo, brand headlines, major headings
```

### Typography Best Practices

```text
✓ Minimum 18px for body text (improved readability)
✓ Line height 1.6 for body text (comfortable reading)
✓ Line height 1.2 for headings (tight, impactful)
✓ Maximum line length: 50-75 characters
✓ Paragraph spacing: 1.5em between paragraphs
✓ Use -0.02em letter spacing for large headings (48px+)
✓ Never go below 14px for any text

✗ Don't use multiple font families
✗ Don't use yellow for body text
✗ Don't use light weights (below 400) for small text
```

## Presentations {{ id: 'presentations' }}

Guidelines for creating UPA-branded presentations (PowerPoint, Keynote, Google Slides):

### Slide Backgrounds

```text
Option 1: Black Background (Recommended)
- Background: #171717 (Brand Black)
- Headings: #FFFFFF (White) or #FFCB05 (Yellow)
- Body text: #E2E8F0 (Light Slate)
- Accents: #FFCB05 (Yellow)
- Effect: Premium, high-end, masculine

Option 2: White Background (Professional)
- Background: #FFFFFF (White)
- Headings: #171717 (Black)
- Body text: #262626 (Black 800)
- Accents: #FFCB05 (Yellow)
- Effect: Clean, professional, versatile

Option 3: Yellow Accent Slides (Impact)
- Background: #FFCB05 (Yellow)
- Headings: #171717 (Black) - BOLD, uppercase
- Body text: #171717 (Black)
- Use sparingly: Key messages, CTAs, section breaks
```

### Title Slide Layout

```text
Layout:
- Full black background (#171717)
- Title: 60px, weight 900, white or yellow, centered
- Subtitle: 24px, weight 400, #E2E8F0, centered
- Logo: Top right corner (if applicable)
- Yellow accent line: 4px thick, #FFCB05, 80% width, below title

Example:
┌────────────────────────────────────┐
│                                    │
│    TRANSFORM YOUR BUSINESS AND     │
│    BECOME THE MAN YOUR FAMILY      │
│           NEEDS                    │
│    ─────────────────────           │ ← Yellow line
│                                    │
│  Ultimate Performance Academy      │
│                                    │
└────────────────────────────────────┘
```

### Content Slide Typography

```text
Slide Titles:
- Font size: 48px minimum
- Weight: 700 (Bold) or 900 (Black)
- Color: White on black, Black on white
- Position: Top left, 80px from top

Body Text:
- Font size: 24px minimum (presenters: 30px recommended)
- Weight: 400 (Regular)
- Line height: 1.5
- Color: #E2E8F0 on black, #262626 on white
- Max 6 bullet points per slide
- Max 8 words per bullet point

Key Points/Stats:
- Font size: 72px minimum
- Weight: 900 (Black)
- Color: #FFCB05 (Yellow)
- Use for emphasis and data
```

### Visual Hierarchy

```text
Level 1 - Main Headline:
- 60-72px, weight 900, yellow or white
- Use for: Slide titles, key messages

Level 2 - Subheading:
- 36-48px, weight 700, white or black
- Use for: Section headers, categories

Level 3 - Body Text:
- 24-30px, weight 400, light gray or black
- Use for: Bullet points, descriptions

Level 4 - Small Text:
- 18-20px, weight 400, light gray
- Use for: Notes, citations, sources
```

### Images & Photos

```text
✓ Use high-quality, professional photos
✓ Black and white photos with yellow overlays
✓ Photos of successful men, business, family
✓ Maintain 30% minimum text contrast over images
✓ Use dark overlay (rgba(0,0,0,0.5)) under white text

Image Treatments:
1. Full-bleed background with text overlay
2. Split screen: Image 50% | Text 50%
3. Image with yellow border (8px thick)
4. Grid of 4 images with yellow dividers

✗ Don't use: Stock photos that look staged
✗ Don't use: Low-resolution or pixelated images
✗ Don't use: Images without proper contrast for text
```

### Data Visualization

```text
Charts & Graphs:
- Primary color: #FFCB05 (Yellow)
- Secondary color: #171717 (Black)
- Tertiary color: #737373 (Gray)
- Background: Match slide background
- Labels: 18px minimum, clear and readable
- Always include data source at bottom

Progress Bars:
- Fill: #FFCB05 (Yellow)
- Background: #404040 (Dark gray) or #e5e5e5 (Light gray)
- Height: 24px minimum
- Border radius: 12px
- Include percentage label

Icons:
- Style: Outline or solid, consistent throughout
- Color: #FFCB05 (Yellow) or White
- Size: 48px minimum for slides
- Use traffic light colors for status only
```

### Call-to-Action Slides

```text
End Slide / CTA:
- Full yellow background (#FFCB05)
- Main CTA: 60px, weight 900, black, uppercase
- Supporting text: 30px, weight 400, black
- Contact info: 20px, weight 600, black
- Optional: QR code (bottom right)

Example:
┌────────────────────────────────────┐
│  Yellow background (#FFCB05)       │
│                                    │
│     START YOUR                     │
│     TRANSFORMATION                 │
│     TODAY                          │
│                                    │
│  Visit: upa-academy.com            │
│  Email: info@upa-academy.com       │
│                                    │
└────────────────────────────────────┘
```

### Animation & Transitions

```text
✓ Simple fade transitions between slides (0.3s)
✓ Bullet points appear one at a time
✓ Stats and numbers count up (for impact)
✓ Subtle entrance animations for headings

✗ Don't use: Spinning, bouncing, or flashy animations
✗ Don't use: Different transitions for each slide
✗ Keep it professional and focused on content
```

### Presentation Best Practices

```text
Content:
✓ One idea per slide
✓ Maximum 6 bullet points per slide
✓ Maximum 8 words per bullet
✓ Use visuals over text when possible
✓ Include data and stats for credibility

Design:
✓ Consistent background throughout (all black or all white)
✓ Yellow accent slides for key messages only (1-2 per deck)
✓ High contrast for all text
✓ White space is your friend
✓ Align all elements consistently

Accessibility:
✓ 24px minimum font size
✓ 4.5:1 minimum contrast ratio
✓ Don't rely on color alone (use icons + text)
✓ Provide text alternatives for graphs
✓ Test readability from back of room
```
